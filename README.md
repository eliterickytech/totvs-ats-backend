
# Totvs ATS

API para podermos fazer cadastrado de vagas, candidatos, atrelar vagas ao candidato.



## Tecnologias implementadas

- ASP.NET WebApi Core 
- FluntValidator
- Swagger UI with JWT support
- Framework 8.0
- MongoDB
- Docker

## Instalação

Instale executando o comando abaixo

```
  docker-compose -up -d 
```
